## Dataset
Spam Mail Dataset

## Technologies
- Python
- Pandas
- Scikit-Learn
- TF-IDF

## Workflow
1. Load Dataset
2. Text Preprocessing
3. TF-IDF Vectorization
4. Train-Test Split
5. Logistic Regression Training
